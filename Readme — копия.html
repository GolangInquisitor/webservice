Тестовый веб сервер


Инструкция


1. Потавить PostgreSQL

2. создать тесовую базу psql -U postgres -d <base> /scripts/postgres/scoltest.sql

3. Сбилдить проект cmd\server_app.main.go

4. Поместить config.yml рядом с исполняемым фалом

5. При необходимости внести правки в настройки соединения в базой данных в config.yml
